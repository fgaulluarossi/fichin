# Fichin
Este es el primer repo de la app familiar  "Fichin"
La idea de la app es tener un intercambio de "Fichines" digitales a modo de "incentivos" por tareas completadas
Se pretende utilizar 2 tipo de usuarios, el usuario administrador o "Jef@ de familia" que es el que tiene la posibilidad de entregar fichines al usuario que realiza la tarea puede ser "hijo,hija,sobrino, etc.".
Cuenta con una interfaz grafica sensilla, que puede ser personalizado por el usuario administrador.
Primer pantalla muestra graficos de "Premios o servicios" que "valen" X cantidad de fichines.
